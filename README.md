# sci-facts

## Setup

1. Install and activate a conda environment named `sci-facts` near the repository (e.g., `conda create -n sci-facts python=3.11 && conda activate sci-facts`).
2. Run `pip install -r requirements.txt` to install the current Python dependencies.
3. Open the repository in the editor; `.vscode/settings.json` already points to the local `sci-facts` interpreter so the workspace defaults to it.
